# Jengo AI Ecosystem

Welcome to the **Jengo AI Ecosystem**, a comprehensive repository and showcase of 14 integrated AI tools, creative utilities, and ecosystem platforms designed for modern builders, creators, and communities.

## 🌟 Overview

From prompt engineering to microtransactions, the Jengo AI Ecosystem explores full-stack creative workflows intersecting with robust AI paradigms. This specific interface has been meticulously designed using a high-contrast, Neo-Brutalist "Editorial Aesthetic"—characterized by its dark canvas, vivid yellow accents, rigorous architectural grids, and beautiful typography.

## ✨ Key Features

- **Dynamic Categorization:** Seamlessly filter the ecosystem via a sticky navigation bar into tailored sectors: *AI Tools, Creator, Social, Business, Payments, and Marketplace*.
- **Editorial Design System:** High-impact, brutalist design utilizing Tailwind CSS. Features deep off-blacks (`#0A0A0A`), high-visibility yellows (`#facc15`), and razor-thin compartmentalization lines (`white/10`).
- **Typography:** An elegant, magazine-style interplay between **Inter** (sans-serif) for technical layouts/labels and **Playfair Display** (serif) for towering, immersive heading elements.
- **Direct Integrations:** Fully functional links to dedicated product pages/monographs (via Selar integrations).
- **Responsive Architecture:** flawless rendering ranging from mobile phones to ultra-wide desktop monitors.

## 🛠 Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS V4
- **Icons:** Lucide React
- **Typography:** Google Fonts (Inter, Playfair Display)

## 📦 Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone or download the repository.
2. Install the exact dependencies required:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```
4. Open your browser to `http://localhost:3000` (or whatever port Vite returns) to view the application.

## 🎨 Design Philosophy
The UI relies deeply on architectural framing and the tension between organic serifs and structural sans-serifs. The grid system enforces constraints that help guide the user visually down the massive 14-app compendium, providing heavy breathing room (padding) while retaining a dense, info-heavy editorial look.
